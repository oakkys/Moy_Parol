Мобильное приложение «Мой пароль»
1. Описание проекта

Приложение предназначено для безопасного хранения логинов, паролей и другой конфиденциальной информации пользователя на мобильном устройстве.

2. Основная цель

Обеспечить быстрый и защищённый доступ к учетным данным с удобным управлением записями.

3. Основной функционал

3.1 Авторизация

- Вход в приложение по биометрии (Face ID / Touch ID)

- Альтернативный вход через мастер-пароль

- Блокировка приложения при отсутствии авторизации

3.2 Главный экран

- Отображение категорий записей:

- Все пароли

- Wi-Fi

- Безопасность

- Удалённые

- Показ количества записей в каждой категории

3.3 Список записей

- Отображение сохранённых аккаунтов

- Название сервиса + логин/e-mail

- Поиск по записям

- Переход в карточку записи

3.4 Карточка записи

- Просмотр данных записи:

- сервис

- логин / e-mail

- пароль (скрыт по умолчанию)

- Показ / скрытие пароля

- Копирование данных в буфер обмена
- Редактирование записи
- Удаление записи
- Переход по ссылке (если есть)

3.5 Добавление записи

- Создание новой записи вручную

- Поля:

- название сервиса

- логин / e-mail

- пароль

- ссылка (опционально)

- категория

- Генератор надёжного пароля

3.6 Удалённые записи

- Просмотр удалённых элементов

- Восстановление записи

- Полное удаление без восстановления

4. Интерфейс

- Тёмная тема оформления

- Минималистичный и современный UI

- Адаптация под разные экраны

- Плавные переходы и анимации

5. Безопасность

- Шифрование всех сохранённых данных

- Локальное хранение на устройстве

- Доступ к данным только после авторизации

- Скрытие паролей по умолчанию
